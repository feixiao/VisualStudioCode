## VSCode格式化代码


### C++
+ [clang-format](./clangFormat/README.md)
+ VSCode配置
    + 下载vscode插件clang-format并配置为默认代码格式化器
    + 生成Google的规范
    ```shell
    clang-format -style=google -dump-config > .clang-format
    ```
